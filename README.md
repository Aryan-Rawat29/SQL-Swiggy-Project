# SQL-Swiggy-Project
Swiggy Sales Analysis — SQL Project
End-to-end SQL analysis on 197K+ Swiggy food delivery records across 28 Indian states and 993 restaurants.
Tools Used
MySQL, Star Schema (Dimensional Modelling)
What I Did

Data Cleaning — Null check, blank detection, duplicate removal using ROW_NUMBER() + CTE
Star Schema — 5 Dimension tables (Date, Location, Restaurant, Category, Dish) + 1 Fact table
KPIs — Total Orders, Total Revenue, Avg Dish Price, Avg Rating
Time Analysis — Monthly, Quarterly, Yearly, Day-wise Order Trends
Location Analysis — Top 10 Cities by Orders, State-wise Revenue
Food Performance — Top Restaurants, Top Categories, Most Ordered Dishes, Cuisine wise Avg Rating
Multi-table JOINs, GROUP BY, Window Functions, CTEs

Dataset
197,430 records | 28 States | 993 Restaurants | Jan–Aug 2025
